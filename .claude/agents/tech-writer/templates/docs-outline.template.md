## Примеры команд
Корректные, проверяемые примеры (источник — STACK/реальный CLI). Например:
- `raxd key create --name laptop` — выпуск API-ключа (показывается один раз).
- `raxd key list` — таблица ключей (id, label, created, last-used).
- `raxd key delete <id>` — отзыв ключа.
- `raxd config port 8443` — настройка порта прослушивания.
- `curl -fsSL <url>/install.sh | sh` — установка.
