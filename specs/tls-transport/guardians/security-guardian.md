## Итог

baseline §1/§2/§4/§6 покрыт полно и проверяемо: §1 (auth — SR-8..13), §2 (транспорт — SR-1..7,
14..16), §4 (аудит/устойчивость — SR-17..21, 24, 25), §6 (Docker — SR-26). §3 (exec) и §5

**Ослаблений baseline ради скорости НЕ обнаружено** (проверено прицельно):
- TLS MinVersion = 1.3 — строже baseline-минимума (усиление), legacy 1.2 не нужен для своей пары клиент↔сервер.
- mTLS — baseline §2 «опционально»; отложен решением дирижёра с компенсацией (TLS1.3+bind+ключ+rate-limit) и эскалацией ОР-1.
- Origin «отсутствует → не отклонять» = буква MCP-спеки (present&invalid→403), не самовольное упрощение; ОР-3.
- Маппинг кодов 401/403/429/501 (SR-13) однозначен, согласован с AC5/AC13; причина отказа только в аудит-лог (анти-enumeration, R4).
- Заявления о контракте keystore (constant-time без раннего выхода, исключение revoked, ErrCorrupt без модификации файла, необратимость Fingerprint) подтверждены guardian'ом по коду — не голословны.
